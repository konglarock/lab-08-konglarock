# Week 9 : Express GPA Calculator API

<b>! First clone this project and run following script below </b>

## available script

`yarn install` <br><br>
install all node package <br><br>
`yarn start`<br><br>
run this project without hot reload<br><br>
`yarn dev` <br><br>
run this project with hot reload

- `DELETE /course/:id`
- `POST /addCourse`
